Top 3 Personal Projects

  • Air Quality Index (Python) The summer before going to college, my roommate challenged me to make an app for the open source smart home, Mycroft - The challenge was a perfect fit, I got to work on using an API to retrieve the air quality in addition to diving headfirst into Mycroft's framework. Overall it turned out pretty well, and to this day I consider the project's comments and documentation to be among my best.
  • Skill Trees (Java) Traveling into the future, during my second semester at RIT, I was running into a problem trying to track my progress in each of the different domains of computer science, and I kept thinking back to video game skill screens which displayed progression of one category as the sum of it's sub-categories and so forth. There's nothing too crazy in the code, but I did write my own Tree data structure with some clean recursion.
  • File Organizer (C++) A file sorting program that would create folders for each given file extension in the starting folder, and then moving each file to it's newly made parent folder. Starting from an Instructables post in python, I first worked to improve the code & style, later on rewriting it in C++ 17, which I was in the process of learning at the time.
